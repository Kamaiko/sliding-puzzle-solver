## Changements appliqués:
- Remplacement de tous les `+` par `╔ ╗ ╚ ╝ ╠ ╣ ╦ ╩`
- Remplacement de tous les `|` par `║`
- Remplacement de tous les `=` par `═`
- Grilles de jeu: `+---+` devient `╔═══╦═══╗` et `├───┼───┤` devient `╠═══╬═══╣`
- Conservation exacte du contenu textuel (pas d'accents, pas d'emojis)
- Conservation de `#` pour la case vide
- Conservation de `->` pour les flèches
- Style identique, juste bordures UTF-8 plus élégantes
